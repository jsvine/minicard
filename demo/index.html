Living on the Edge
The .minicard-bleed class removes the margins from the .minicard-body.
.minicard-foot divs are optional, and can be used with both standard minicards and .minicard-bleed minicards.
Helper Classes

The .minicard-row and .minicard-col classes are optional, but useful for laying out multiple cards.

A simple CSS media query can make these columns "responsive." See the final lines of demo.css for an example.
